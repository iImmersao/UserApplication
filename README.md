# User Application (Spring Mimic Example)

This is a sample Java application built using the **Spring Mimic** framework — a lightweight, annotation-driven framework inspired by Spring Boot.

It demonstrates:

- CRUD operations on a `User` entity
- Use of dynamic repository interfaces
- Dependency injection and component scanning
- Authentication and authorization
- Pagination, filtering, sorting
- Basic integration with MongoDB / MySQL / H2

---

## 🚀 Getting Started

### 1. Clone and Build the Framework

```bash
git clone git@github.com:iImmersao/SpringMimic.git

Then open the project with Intellij and run the Maven install target.

git clone git@github.com:iImmersao/UserApplication.git

The open the project with Intellij, open Main.java, and run.

```
